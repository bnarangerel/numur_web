File Structure

Our theme file structure that looks like this.

  • 📁 dist - Generated all files
  • 📁 docs
    • 📁 assets
      • 📁 css - Compiled CSS
      • 📁 fonts - All fonts
      • 📁 images - Image assets
      • 📁 js - Javascript source
    • 📁 pages - HTML pages
  • 📁 fonts - All fonts
  • 📁 js - Javascript source
  • 📁 libs - All library
  • 📁 node_modules - Directory where npm installs dependencies.
  • 📁 scss - SCSS source for theme
  • 📄 gulpfile.js - All the build commands
  • 📄 package.json - List of dependencies and npm information
